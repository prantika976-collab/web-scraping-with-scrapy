# Web Scraping with Scrapy

## Author

**Prantika Biswas**

## Project Overview

This project demonstrates the end-to-end process of building a web scraper using Scrapy. The scraper extracts structured book information from the Books to Scrape website and exports the collected data into JSON format for further analysis.

The project was developed as part of a project-based learning journey to strengthen practical skills in Python, web scraping, data extraction, and workflow automation.

## Features

* Scrapy project setup and configuration
* CSS selector-based data extraction
* Scraping book listings
* Following links to detail pages
* Extracting detailed book information
* Handling pagination across multiple pages
* Large-scale crawling of website content
* JSON data export
* Structured and reusable scraping workflow

## Technologies Used
